# Funda

**Funda**는 개발자라면 반드시 알아야 할 컴퓨터 과학 지식 공부를 게임처럼 바꾸고, 자동 복습으로 꾸준히 기억을 쌓게 만드는 웹 기반 학습 플랫폼입니다. 
레벨 업 · 캐릭터 성장 · 데일리 세션으로 습관형 학습을 유도하며 개인 별 약점을 자동으로 보완해줍니다.

## 💻 핵심 기능

- **퀴즈 세션**: 짧은 라운드 위주, 여러 유형, 즉각 피드백·간단 해설
- **SRS 복습 큐**: 틀린 문제는 빠르게, 맞춘 문제는 간격을 두고 자동 재등장
- **캐릭터·보상**: 문제 풀이 → XP → 레벨 업 → 캐릭터 성장/이펙트, Streak · 데일리 목표로 루틴 생성
- **마이페이지**: 레벨/ XP/ streak, 학습량, 강점/약점 영역 시각화, “오늘의 추천 학습” 진입
- **개인화**: 정답률 · 개념별 약점 분석 기반 난이도/문제 구성 자동 조정
- **경쟁 시스템**: 할당된 그룹에서 상위권을 차지할 경우 레벨 상승

<br/>

## 🚀 Getting Started

### 1. 프로젝트 클론

```bash
git clone https://github.com/boostcampwm2025/web21-funda.git
```

### 2. 의존성 설치 및 실행

> Note: 이 프로젝트는 pnpm을 사용하여 모노레포를 관리합니다.
> 

```bash
# 의존성 설치 (루트 디렉토리에서)
pnpm install

# 개발 환경 실행
pnpm run dev
```
